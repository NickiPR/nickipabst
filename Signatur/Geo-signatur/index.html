Step 1:
Choose the language and department in the menu
Step 2:
Edit directly in the shown signature. When you are done press Generate and copy the code.

Step 3:
Open Notepad and paste the text onto a new document. Save the document and write “.htm” (not html!).

Step 4:
In Outlook, go to File > Options
From the Outlook Options dialog, click Mail.
On the right side press CTRL and click the Signatures

This will open the apps folder that contains your Microsoft Outlook signatures. Usually, it is the C:\Users\initials\AppData\Roaming\Microsoft\Signatures folder.

Step 5:
Copy your .htm file into this folder.

Close the folder and return to Outlook Options.
Click OK.

Repeat these 5 steps for each signature you need

Double check if it is working
Your newly added HTML email signature should show in your Signature drop-down list. You should go to the Outlook’s Signatures and Stationery dialog and make sure you’ve assigned this to the proper account and if it should apply to new messages as well as replies and forwards.
